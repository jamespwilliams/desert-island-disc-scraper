# desert-island-disc-scraper

## Usage

Python 3 is needed.

```console
git clone https://github.com/jamespwilliams/desert-island-disc-scraper.git
cd desert-island-disc-scraper
```

If using `nix`:

```console
nix-shell

python scraper.py
```

Otherwise:

```console
. venv/bin/activate
pip install -r requirements.txt

python scraper.py
```
